Agrosour

Agrosour is an innovative mobile solution designed to assist farmers with precision agriculture by utilizing sensors, AI, and real-time data to optimize crop yield, soil health, and overall farming efficiency. Learn about global natural occurance through Nasa.

Current Features

  • User Profile for personalized tracking
  • Google Authentication for secure sign-in
  • Real-time Data Collection from NPK Sensors
  • Soil Moisture and Temperature Monitoring
  • AI-based Leaf Disease Detection
  • Weather Forecast Integration
  • Flood Detection and Alerts
  • Community Features for sharing insights
  • Global Natural Event Showcase via NASA API

Dataset/API Used

EONET (NASA Earth Observatory Natural Event Tracker)

Offers real-time data on natural events like floods, fires, and environmental changes that impact farming conditions.

Asia Irrigation Only (No Groundwater Depletion)

Focuses on irrigation activities in Asia, helping farmers with water management based on regional trends.

Asia Groundwater Depletion Only (No Irrigation)

Provides groundwater depletion data for Asia, helping to address over-extraction concerns and manage water use more effectively.

OpenWeather API

Delivers localized weather data such as temperature, humidity, wind speed, and rain probability for real-time weather forecasting.

Open-Meteo API

Provides free weather forecasts, including rainfall, temperature, and humidity, used to inform irrigation suggestions and field conditions.
